   • Auto-login - Restaura sesión al abrir la app
   • Manejo de errores - Mensajes informativos en UI
   • Logout seguro - Limpieza completa de credenciales

   FLUJO:
   LoginScreen → ApiService.login() → POST /api/auth/login/
            ↓
        Recibe token → SharedPreferences.setString('token', ...)
            ↓
        AuthProvider.isAuthenticated = true
            ↓
        Redirige a HomeScreen
            ↓
        Token automáticamente inyectado en Authorization header

     - ADMIN (isStaff = true) → Acceso completo a panel administrativo

   • Validación en RoleProtectedRoute - requireAdmin parámetro
   • Protección en backend - Endpoints Django con permisos
   
   EJEMPLO:
   RoleProtectedRoute(
     requireAdmin: true,  // Solo ADMIN
     child: AdminParcelasScreen(),
   )
