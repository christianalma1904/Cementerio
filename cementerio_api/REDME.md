# Cementerio API

Backend en Django + Django REST Framework conectado a PostgreSQL para la gestión de:
usuarios, parcelas, reservas, pagos y difuntos.

## 1. Requisitos

- Python 3.11
- PostgreSQL
- pip / virtualenv

## 2. Instalación

```bash
git clone <repostorio>
cd cementerio_api
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
